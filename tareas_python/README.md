# Simple CRUD API - Node.js

API REST básica para gestionar tareas usando Node.js, Express y PostgreSQL.

## Tecnologías

- Node.js
- Express
- PostgreSQL

## Endpoints

GET /tareas  
Obtiene todas las tareas

POST /tareas  
Crea una tarea

PUT /tareas/:id  
Actualiza una tarea

DELETE /tareas/:id  
Elimina una tarea

## Instalación

1. Clonar el repositorio

2. Instalar dependencias

npm install

3. Ejecutar el servidor

node server.js
